Design Theme
My portfolio has a simple and clean design because I am still learning HTML and CSS. I wanted the website to look neat, easy to read, and not overly complicated. I used light colors and clear fonts so that the website feels friendly and welcoming to visitors.

Challenges Faced
One challenge I encountered was adding links in the HTML document that would redirect to my GitHub account and the live demo of the project. To solve this problem, I went back to my GitHub account, downloaded a new ZIP file of the repository, and ran the files again.

Another challenge was deciding when to use Flexbox and when to use CSS Grid. While building the layout, I realized that Flexbox works better for simple one-direction layouts such as the navigation bar, footer, and skills list.

I also faced a challenge when trying to run the HTML file. The browser kept showing an error when I clicked the Run and Debug button, and the file would not load. To fix this, I located the file in the folder, right-clicked it, and opened it using Live Server, which allowed the page to load correctly.
